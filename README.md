# Installation

## Installing Laravel

###### Via Composer Create-Project

** Create project laravel by composer:**
```bash
composer create-project --prefer-dist laravel/laravel name_project
```

** Local Development Server **
(access project and execute command below)
```bar
php artisan serve
```
## Configuration
